\end{enumerate}

The shape is a property of the numpy array, it's used to get the current shape of an array.

\end{enumerate}

\begin{itemize}
    \item \texttt{np.add()}
    \item \texttt{np.subtract()}
    \item \texttt{np.multiply()}
    \item \texttt{np.divide()}
    \item \texttt{np.power()}
    \item \texttt{np.reminder()}
